# 🍽️ Restaurant Management — Cahier des Charges

Application complète de gestion de restaurant avec **tarification dynamique par palier de quantité**.

## 📋 Contenu du Projet

| Fichier | Description |
|---------|-------------|
| [cahier_des_charges_restaurant.md](cahier_des_charges_restaurant.md) | Spécifications fonctionnelles et techniques complètes |

## 🎯 Fonctionnalités Clés

- **Tarification dynamique** — Prix unitaire variable selon la quantité commandée (paliers configurables)
- **Gestion complète** — Menu, commandes, tables, facturation, cuisine (KDS)
- **Multi-rôle** — Admin, Serveur, Cuisinier, Caissier, Client
- **Commande en ligne** — QR code, paiement Stripe, suivi en temps réel

## 📊 Priorités

- **MUST HAVE** — 25 fonctionnalités obligatoires (menu, commandes, tables, facturation, KDS, auth)
- **SHOULD HAVE** — 19 fonctionnalités recommandées (en ligne, stocks, réservations, fidélité, rapports)

## 🚀 Phases

1. **MVP** — Menu + tarification + commandes + tables + facturation
2. **Cuisine & Rôles** — KDS + authentification + rôles
3. **En ligne** — Commande client web + paiement en ligne
4. **Avancé** — Stocks + réservations + fidélité + rapports
